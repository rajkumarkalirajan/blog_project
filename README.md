\# Blog Project Deployment (Docker + AWS)



This project demonstrates deploying a Django application using Docker and AWS EC2 with Amazon ECR.



\---



\## 🚀 Features

\- Dockerized Django app

\- Hosted on AWS EC2

\- Image stored in Amazon ECR

\- Accessible via public IP
